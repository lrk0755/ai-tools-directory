# AI Tools Directory - 项目规格说明书

## 1. 项目概述

- **项目名称**: AI Tools Directory
- **类型**: Web应用 (Next.js)
- **核心功能**: 展示GitHub热门AI工具的目录网站，实时获取热门项目
- **目标用户**: 开发者、AI爱好者、技术人员

## 2. UI/UX 规格

### 布局结构

- **Header**: 固定顶部，包含Logo + 搜索框 + 导航
- **Hero**: 简短介绍区域
- **工具网格**: 响应式卡片网格展示工具
- **Footer**: 版权信息

### 响应式断点
- Mobile: < 640px (1列)
- Tablet: 640px - 1024px (2列)
- Desktop: > 1024px (3-4列)

### 视觉设计

**色彩方案** (暗色主题 - 科技感)
- 背景: `#0a0a0f` (深黑蓝)
- 卡片背景: `#12121a`
- 主色调: `#6366f1` (Indigo)
- 强调色: `#22d3ee` (Cyan)
- 文字主色: `#f8fafc`
- 文字次色: `#94a3b8`
- 边框: `#1e1e2e`

**字体**
- 标题: Inter, system-ui
- 正文: Inter, system-ui
- 代码/数字: JetBrains Mono

**间距**
- 卡片内边距: 24px
- 卡片间距: 20px
- 页面边距: 24px (移动端), 48px (桌面端)

**视觉效果**
- 卡片hover: 轻微上浮 + 发光边框
- 按钮hover: 亮度提升
- 加载: 骨架屏动画

### 组件

1. **工具卡片**
   - 图标/头像
   - 标题
   - 描述(截断)
   - 星标数 + Fork数
   - 编程语言标签
   - 分类标签

2. **搜索栏**
   - 输入框 + 搜索图标
   - 实时过滤

3. **分类筛选**
   - 可选按钮组

4. **排序选项**
   - 星标数 / 最近更新 / 名称

## 3. 功能规格

### 核心功能

1. **数据获取**
   - 从GitHub API获取 `topic:ai` 热门仓库
   - 支持分页
   - 缓存机制避免API限流

2. **展示列表**
   - 显示: 仓库名、描述、星标数、Fork数、语言、更新时间
   - 支持点击跳转到GitHub

3. **搜索功能**
   - 按名称/描述搜索
   - 实时过滤

4. **分类筛选**
   - 预设分类: LLM、图像生成、自动化、开发工具、教育
   - 通过仓库的topics自动判断

5. **排序**
   - 按星标数 (默认)
   - 按更新时间
   - 按名称字母

### 数据处理

- 首次加载从GitHub API获取
- 本地缓存数据到JSON文件
- 定时刷新(可选)

### 边界处理

- API限流: 显示缓存数据 + 提示
- 无结果: 显示空状态
- 加载中: 骨架屏
- 错误: 重试按钮
